# ofRoundBunny

ROUND ROUND BUNNY.

Around the rabbit to music.

## Environment
* OS X 10.11.3
* OpenFrameworks v0.9.3
* Syphon server
* Syphon client

## Requirement
* OpenFrameworks v0.9.3
* Sython
* ofxSyphon

## Installation



    $ cd of_{version > 0.9.3}/apps/yourApps/
    

    $ git clone https://github.com/6uclz1/ofRoundBunny/
